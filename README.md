## 🚀 시작 어떻게 하냐고?

```bash
# 배포했으니까 들어가
https://meow-delta-eight.vercel.app
```

로컬 환경에서 보고싶으면,

```bash
# 이 프로젝트 땡겨
git clone https://github.com/your-username/MEOW.git
cd MEOW

# 뭐 설치하라니까 해
npm install

# 개발 서버 킨다
npm run dev
```
